Academic Honesty as it relates to the school Honor Code

Some of you may have experience with HTML/CSS and other web technologies in the past. One of the things you have to be pay attention to is the Academic Honesty. Because we have students from a lot of countries, I want to make sure you all understand what BYU-Idaho expects. Each culture has a slightly different interpretation of these Honesty Policies and in the work place it is different as well, but we are in a school setting.

This is taken directly from training about the various ways students can violate the Honor Code. In this class, and many of the development classes, the most common forms are Fabrication, Cheating, and Other Misconduct from the definitions below.

The way you cheat is by using code you did not write, using libraries, such as jQuery, and using templates or frameworks such as Bootstrap without authorization from the instructor to complete the work. Some libraries are required to complete work in other classes. The real world this is not an issue to use public libraries as that is what they are for. For school work, we want to make sure you know how the code works by having you create it from scratch. This does help you once you get into the real world to troubleshoot bugs.

Academic Dishonesty Definitions

Inadvertent Plagiarism

Inadvertent plagiarism involves the inappropriate, but non-deliberate, use of another's words, ideas, or data without proper attribution. Although not a violation of the Honor Code, it is a form of academic misconduct for which an instructor can impose appropriate academic sanctions.

This type of plagiarism most often occurs when another student is helping your learn the code and you end up copying it. This is not allowed. When helping each other, you must remember that you cannot copy the HTML code and text directly for the entire page. It is normal for some parts to be the same but not everything.

Plagiarism

Intentional plagiarism is the deliberate act of representing the words, ideas, or data of another as one's own without providing proper attribution on the original author through quotation, reference, or footnote.

Plagiarism is probably the most well-known element of academic dishonesty. This includes the use of artificial intelligence technologies such as ChatGPT, Copilot, etc or using HTML, CSS, or JavaScript you did not personally write and passing it off as your own.

Fabrication or Falsification

Fabrication or falsification occurs when a student invents or distorts the origin or content of information used as authority. Examples include:

  • Citing a source that does not exist

  • Citing information from a source that is not included in the source for which credit is given

  • Citing a source for a secondary proposition that it does not support

  • Using code from another site without giving source credit to the original author.

Cheating

A student cheats when he or she attempts to give the appearance of a level of knowledge or skill that has not been obtained. Examples include:

  • Using unauthorized materials during an examination or while completing an assignment, this includes artificial intelligence technologies such as ChatGPT

  • Collaborating on an examination or assignment without authorization

  • Taking an examination or completing an assignment for another, or permitting another to take an examination or to complete an assignment that is not his or her own

  • Using code you did not write to speed up development
  • Using a library without permission
  • using a Framework without permission

Other Misconduct

Other academic misconduct includes other academically dishonest, deceitful, or inappropriate acts which are intentionally committed. Examples include:

  • Obtaining or providing to another a test or answers

  • Submitting the same work for more than one class without disclosure and approval. There are many classes you can take where you may work on your resume/CV. If you have worked on it in another course your instructor needs to know that is where you are starting from and not creating it from scratch. Your instructor will give you guidance as to what to do.

  • Getting equal credit on group assignments when equal work was not done

  • Using content from a previous/other class in your current class.

  • Using artificial intelligence technologies to help or do your work as a student. The list of AI tools is too long to list all of them. Just know that you are not allowed to use them for anything that you submit for grading.

Additional information about what is allowed

I get this question all the time: Are Libraries and Frameworks allowed in this class: The short answer NO

Do not use CSS frameworks in part or full. Students have tried to use individual rules from Bootstrap, Foundations, and Material CSS in my class. Do not do it, I have been using these myself in my profession so I recognize them very quickly. Frameworks are also detected through the use of the Page Evaluation as they will make the total size of the pages too large for the requirements of schoolwork. You will get to use them in a future class.

I will allow you to use icons from Hero Icons ( https://heroicons.com ) and Box Icons ( https://boxicons.com ) for icons to be used in your final project. You must copy the SVG code of the icons into your code or make them images you reference. You will not be permitted to use them as font icons. This means no Font Awesome allowed. Font Awesome does not pass CSS validation and you will loose points for that, so I do not allow it.

There is not JavaScript allowed in this class.

If any of you have questions about this please let me know.
