Most of you probably have the Microsoft Teams app on your computer but it will not work with the logging into the school system. When you launch the app and you see the button that is in this screenshot, you must click it and install the New Teams application to be able to log into the school account. If you are running a version of Windows previous to Windows 10, you will not be able to upgrade to the new Teams application. If you run into this issue you will have to use the Web Based interface but going through Outlook to load it as described below.
