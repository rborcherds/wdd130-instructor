Learning Activities

This week we will be doing the following learning activities. After you do the learning activities you will then take a quiz on what you have learned. You are allowed to retake the quiz a maximum of 3 times and the latest score is what will be recorded.

  • You will then learn about Code comments - I will give you an example of a CSS file with comments in it that I use when writing a straight CSS file in Week 4
  • You will then learn about CSS values, and units of measurements. I do not want you to use % until you understand how it works. Use px for the majority of you sizing and use em and rem for the fonts. You can use em and rem for sizing but that takes a little bit of math skills to know exactly what size they are. There are quite a few you will learn about. The ones you will use most of the time are px, rem, rem and %.
  • You will learn more about float as you will be required to use that this week.
  • You will then learn about positioning. You will have to use this in this week's assignment you submit for grading
  • You will also learn more about CSS variables. You must use these on all web pages you will be creating going forward in this class.

 

Code Along Activity

This week's code along activity you will start to use CSS Selectors. The type of selectors we will cover in this class are:

  • Universal
  • Type - These are the HTML tag names
  • Class - This is what you must use for all CSS that are not a tag name
  • ID - Do not use these in your CSS. 

There are quite a few other selectors that we will not cover in the course material but you are free to learn about here:  https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Selectors and this one goes with it as these combinators create complex selectors: https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Selectors/Selectors_and_combinators

You are not allowed to use CSS code directly in the HTML files. You must use a CSS file that you link too to be HTML5 compliant.
