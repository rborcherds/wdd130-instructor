Here is the CSS I used for the details and summary tags with the ::after pseudo tag for the chevron to rotate 90 degrees when you open the details option. You will notice that I kept the CSS as simple as possible. All of the HTML tags could be converted to class names. I do not use ID names in my CSS files. The only time ID is used for CSS is when I am having JavaScript apply some CSS to that element. If I have to apply it to multiple then I do use the class name in JavaScript.

To calculate the decimal to hex it uses this format:

0 to 9 is the same number 

10 = A

11 = B

12 = C

13 = D

14 = E

15 = F

16 = 10

all the way to 255 = FF

This is true for the 3 numbers represented as RED, GREEN, BLUE.

Most computers have a calculator build into them that can do the conversion for you automatically.

The transparency uses this to get a decimal number to use.   255*( % of transparency ) = decimal number. That is converted to the HEX number.

255 * .7 =  178.5. You drop the decimal or you round up and use 178/179 which = B2/B3 depending if you round up or not.
