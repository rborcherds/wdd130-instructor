Font styling you will learn about this week

font-family  - This is the font we use for specific text.

font-style  - This is normal, italics, or oblique which leans the text a little more than italics

font-size  - This is how big you want the text denoted as px, em, or rem

font-weight  - is how you set bold for example.  The font weight can be a number from 100 to 900, in 100 increments, if the font supports it. Not all fonts support all weights. You have to know the font. 100 is very thin lines where 900 is the thickest lines for each letter.  Bold is normally 700. Normal is 400

font-variant  - This allows you to use small-caps on text. All letters are uppercase but the first letter of each word is normal size and the rest of the word is uppercase but the height of lowercase letters.  https://www.w3docs.com/tools/editor/153
