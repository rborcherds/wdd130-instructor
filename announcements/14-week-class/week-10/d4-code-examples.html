These links to code examples from requests that students have made in previous semesters.

This first one lets you play with color hues. The way it works is you adjust the CSS --color and --l variables. This uses the HSL color codes. You can use this website to get the numbers.   https://hslpicker.com/   Look at the HSLA color and only use the first 3 percent numbers. You put the first 2 in the --color variable and the last percent number into the --l variable.

The Lorem Ipsum text is 15% darker and the background is 15% lighter than the starting color.

https://borcherds-ralph.github.io/wdd130/examples/color-hue.html

This last one I did not create but I wanted to point it out. This uses Flex to display all the various input types for forms. A lot of them require JavaScript to work correctly. Use this to see what the various input options are and get an idea of how to use them.

https://codepen.io/ralitsavoronevska/pen/yRGjvQ

This does use the Bootstrap CSS Framework for a lot of the styling along with 6 other CSS files from other sites. Click on the gear icon in the CSS section (top right of the box) and you can see the URL's for the other CSS files. There are 8 JavaScript libraries used along with the custom code to make some of these work. You will notice that Bootstrap has JavaScript library. This is one of the reasons I do not like Bootstrap, you have to load their JavaScript library to make a lot of things work in Bootstrap. This makes it a lot more complicated to work with. There are 3 JavaScript libraries loaded in the HTML config area as well.
