Quite a few students want to use YouTube videos. If you use the code that is given to you by YouTube to embed the video there are a couple of things to be aware of.

  1. The width and height are preset in the code.
  2. There is an invalid property that fails validation

The code looks something like this. The issues are in RED and BOLD

<iframe src="https://www.youtube.com/embed/videoID" title="YouTube video player" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen="" width="560" height="315" frameborder="0"></iframe>

These two items need to be move to the CSS file. The way to do is is to modify this code slightly and remove those items and put a class name. I have put the class is BOLD and BLUE

<iframe src="https://www.youtube.com/embed/videoID" title="YouTube video player" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen="" class="youtube"></iframe>

Here is the CSS rule for the class. If all your videos are the same size, you can create 1 rule for all of them. You can leave the size if you like but the frameborder must be moved to the CSS.

.youtube {
    width: 560px; 
    height: 315px; 
    border: 0;
}
