Grid Layout

A lot of you are having issues with Grid layout. Most are missing basic concepts about grids.

A few rules to remember

  • column/row numbers refer to the starting line of the column/row. A grid-column: 2/5; means that you start at column 2 starting line and end at the starting line for column 5.
  • Even if you define the rows and columns, you can always add rows/columns outside of the defined grid. This does change the layout.
  • Using grid-column-start, grid-column-end values is easier than grid-area.  
These two properties
grid-column-start: 2;
grid-column-end: 5;

are the same as this
grid-column: 2/5;

and the same as this grid-area
grid-area: 2 / 5 / 1 / 1;


These 4 properties
grid-column-start: 2;
grid-column-end: 5;
grid-row-start; 1;
grid-row-end; 6;

are the same as this
grid-column: 2/5;
grid-row 1/6;

and the same as this grid-area
grid-area: 2 / 5 / 1 / 6;

 

There are a lot of ways to define columns and rows. These are just the basics.
