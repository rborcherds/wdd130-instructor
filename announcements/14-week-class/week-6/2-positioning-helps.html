Make sure to read the hints for each activity as there is some notes that the creators wrote. There is a Discussion Forum post that gives more hints as well.

  • All items except #2 & #6  must go full width inside the borders already set.
  • #2 width and height of the colored boxes cannot be changed. You use what you have been give to do the layout shown in the example image
  • #5 must be able to keep the text in the yellow at all time no matter with width of the screen. Chances are that your screen size is different from mine and that does affect the layout.
  • #6 you do need to look at the image in the hint as it shows you the grid lines. You must also change the width and height of each color to 100% and set the grid size correctly in the content6 rule. This is a 6x6 grid of equal width and height for each row and column.

 

How I grade this assignment

The CSS is given to you to start with. I copy your CSS from the top to this point.This is usually around line 210 in your CSS file. 

/* Do not make any changes below here */

Any changes after these two lines with is line 160/161 in the CSS give to you is not allowed. I have had a lot of students in the past that have set a fixed width to do the layouts past this line thinking that I would not notice. This is why I grade in this manner.
