Positioning Activity

I want you to all start thinking of web pages as dynamic in width. This assignment can help with that. Here are some tips.

 

For some reason a lot of students do not use the hints in this activity.

Every single activity in this assignment has a hint on how to accomplish this.

  • Click the bar with the word HINT in it.
  • Only activity 1 has instructions that are given to you to do this first one. Everyone should get this correct if you follow the steps exactly.
  • Each one has a link to an image as to how it is supposed to look.
  • Activity #6 hint has an additional graphic in the hint with the grid lines to help you with the layout.

 

Activity #2 can be done with either grid or margins. Do not use both as I will have you redo it.

Activities 3, 4, & 6 should have the width and the height set to 100%.   Set the height  in the .content rule for each and use 1fr measurements for the width.  That make them dynamic width. Only #6 has a set width and height set in the parent rule only.

If you do not you can end up with something like this on my monitor.  I use a 3525px wide monitor for grading. It shows when something is not dynamic very quickly.
