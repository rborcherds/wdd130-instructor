This final video shows how to finish up the last page and as usual using class names form previous page where possible.

In this video I use some fairly new HTML tags called <details> and <summary>. These 2 tags work together to give you a collapsible sections.

 

 

 

Here is the CSS for the FAQ page, just the details, summary and other info as described in the video.

details {
    grid-column: 2/12;
    background-color: #496bac;
    border-radius: 30px;
}

details summary {
    font-size: 1.25em;
    padding: 1em;
    background-color: #933503;
    font-size: 2em;
    font-weight: 700;
    letter-spacing: 1px;
    display: flex;
    justify-content: space-between;
}

summary:hover {
    background-color: #1b1b1b;
}

summary::after {
    content: '\276f';
    width: 1em;
    height: 1em;
    text-align: center;
    transition: all 0.35s;
}

details[open] summary::after {
    content: '\276f';
    width: 1em;
    height: 1em;
    transform: rotate(90deg);
}

details h3 {
    font-style: italic;
    padding: 0 3em;
    font-size: 1.5em;
    color: #1b1b1b;
}
details p {
    padding: 1em 2em;
    line-height: 1.5;
    font-size: 1.25em;

}

details img {
    border: 8px solid #a58662;
    border-radius: 30px;
    padding: .5rem;
    background-color: #1b1b1b;
}

details cite {
    font-size: .75em;
    font-weight: 300;
    font-style: italic;
}

.quote {
    font-size: .9em;
    font-weight: 300;
}

.color-text {
    color:#1b1b1b;
}
.color-text:hover {
    color:#a58662;
    text-decoration: underline;
}
