This does not list all the selectors, for example it does not go through all the pseudo-class selectors. This URL give you all the pseudo-classes  that are available.

Pseudo-classes

https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-classes

 

By using the attribute selectors you can have a lot more selectors to use. For example in the Dry Oar website you can use the following from an alt= of and image 

[alt="running the rapids"] {  width: 75%;}

This will apply to any image that has the alt="running the rapids" description on it.

Another way to use this is for a href links that open in a new tab such as the social media icons in our site. This will only affect links that open in a new tab/window. 

[target="_blank"] {
    box-sizing: border-box;
    padding: 1em;
}
