I was asked by one of the students how you could add a 4th page to the site map area of the Site Plan you are working on for your personal site.

They want to do something like this

 

To do this you have to modify the grid-template-area definition and also increase the columns from 6 to 8 columns.

Add 3 new rules and 3 new HTML div statements.

You can see the modified page I did here: https://borcherds-ralph/wdd130/examples/site-plan.html

Here is the CSS for this section. Again the new rules will be bold and the updates will be in BLUE 

.sitemap {

          display: grid;

          justify-content: center;

          grid-template-columns: repeat(8, 12%);

          grid-template-rows: 3em 1.5em 1.5em 3em;

          grid-template-areas: ". . . home home . . ." 

        ". . . . top . . ." 

        ". left left center1 center2 right right ." 

        "page2 page2 . page4 page4 . page3 page3";

    }

        .sitemap > div {

          text-align: center;

    }

        .sm-home {

          grid-area: home;

          background-color: #999;

          line-height: 3em;

    }

        .sm-page2 {

          grid-area: page2;

          background-color: #999;

          line-height: 3em;

    }

        .sm-page3 {

          grid-area: page3;

          background-color: #999;

          line-height: 3em;

    }

        .sm-page4 {

          grid-area: page4;

          background-color: #999;

          line-height: 3em;

        }

        

        .top {

          grid-area: top;

          border-left: 1px solid;

    }

        

        .left {

          grid-area: left;

          border-top: 1px solid;

          border-left: 1px solid;

    }

        .center1 {

          grid-area: center1;

          border-top: 1px solid;

        }

        .center2 {

          grid-area: center2;

          border-top: 1px solid;

          border-left: 1px solid;

        }

        

        .right {

          grid-area: right;

          border-top: 1px solid;

          border-right: 1px solid;

    }
