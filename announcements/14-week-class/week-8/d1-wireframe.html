At the bottom of the screen there is a plus sign that you click on to add a new page.

The text that states Page-1, you can rename by clicking on the up chevron (up carrot/arrow) and select Rename

 

 

 

To download the application to install on your computer go to https://github.com/jgraph/drawio-desktop/releases and download the software for your system and install it. If you have questions about that software please ask. Where you see (version), this is the current version number. This changes every time a new version is released.

Windows installers have this in the name: draw.io-(version)-windows-installer.exe or draw.io-(version).msi
Mac OS use names like this: draw.io-universal-(version).dmg

Linux:  

Ubuntu:   drawio-amd64-(version).deb
App Image for all distros:   drawio-x86_64-(version).AppImage
Redhat/Fedora:   drawio-x86_64-(version).rpm
