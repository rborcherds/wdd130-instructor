Folder and File structure

This is what you should have by the end of week 4. 

  • 🗁 aboutme
    • 🗁 images
    • 🗁 styles
    • 🗎index.html
  • 🗁 challenge
    • 🗎This is were you can save your challenge assignments from week 4, 5 6, 8 & 9
  • 🗁 practice
    • 🗎 This is where you should have the practice assignments from week 2, 3, & 4.
    • 🗎All these weeks create 1 page with the CSS. You can then apply this CSS to the
      wdd130/index.html file to format the homepage
  • 🗁 wwr
    • 🗁 images
      • 🗎all your images
    • 🗁 styles
      • 🗎 styles.css
      • 🗎 site-plan-rafting.css
    • 🗎 index.html
    • 🗎 site-plan-rafting.html
  • 🗎 index.html

The aboutme folder should contain the index.html file for your week 1 Getting to Know You discussion board post, the images folder with your image and if you have done any CSS then the styles folder with the CSS file in that.

The wwr folder will have the folders for your images ans stylesheets and the index.html and site-plan-rafting.html

wdd130/wwr/styles/site-plan-rafting.css    <- Week 2
wdd130/wwr/styles/style.css                <- Week 4

 

You should have a  practice folder. That is where you must put the practice files you  have been working on from Week 2 to Week 4. The practice folder should contain the page for the BYU-Idaho HTML and CSS file. You can take the CSS from the practice assignmetns and apply that CSS to your wdd130/index.html file to format your repository homepage as I did here: https://borcherds-ralph.github.io/wdd130/. I did not use all the CSS but you get the idea.

 

You should create a  challenge folder for assignments that are not submitted for grading in week 4, 5, 6, 8, & 9. 

There will be a lot of small details for you to do. Pay close attention to the details.

Let me know if you have any questions about this structure.
