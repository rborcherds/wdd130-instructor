HTML & CSS styling for your Home Page

This week you will start styling you Home Page with CSS. It is very important that you have finished the Home Page assignment before starting this one. Every web page you visit on the internet uses CSS to style the way the page looks and acts most of the time. Sometimes it requires JavaScript with CSS to make a page act a certain way.

  • The first thing you do is watch 3 video and take a quiz on those videos.
  • You then do an activity on Units of Measurement Practice . This is a very important activity in that you learn various ways that items are measured in a web page. Thee are relative and absolute measurements in web development.
  • You then have an activity where you learn about Block vs Inline elements . Block is the default layout. Inline was added shortly after block to the way a page layout can be done. You have to now how these work because if you do not specify a display type, you will default to block.
  • You then have a CSS Selectors Practice activity. For this class you will use global, tag name, class and sometime ID. You will also use pseudo selectors. There are more types but you will not really get into those during this class.
  • You will have a Challenge activity. You can work in groups if you wish on this activity. There are 2 images in the challenge but it is only one page. The title of the page has a hover effect when you move the mouse over the title of the page.
  • You have an activity about Backend Development . You watch a video about what they do and some of the technologies they use and the type of career options are out there for these types of developers.
  • You then do your assignment to submit: Adding More HTML and CSS Remember to look at the rubric and to follow the instructions carefully. This is when a lot of students start to miss points because they did not follow the instructions carefully.

Important info about adding styling to your site .

  • It is vital that you DO NOT USE the site-plan.css file for any other work.
  • All styling for the website must be in a style.css file.
  • DO NOT USE styling within the HTML file. All styling must be done in the CSS file.
  • Remember that the colors and fonts you chose in the site plan are the colors and fonts you are to use for this site. If you change colors, fonts or sub pages you must update the site plan files. Final grade compares your site to the site plan to see how well you followed the site plan.

 

Spiritual Thought/Question for the Week

In the scriptures we read that we are to have a Broken Heart and a Contrite Spirit ( 2 Nephi 2:7, 3 Nephi 9:20, Moroni 6:2 ).

The meanings of these two terms is very important for us, especially as it relates to school.

Broken Heart : Being humble and under proper control of ourselves.

Contrite Spirit : Contrite means desiring correction as needed

For us to learn effectively, we need to be humble and not proud so that we are teachable and we need to have the desire to be corrected when we do something incorrect. This is how we learn and grow the most effectively.
