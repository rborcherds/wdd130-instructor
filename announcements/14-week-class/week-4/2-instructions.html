It has come to my attention as I am looking at your CSS files to see if you are making all the correct changes, I am seeing a few things that need to be clarified.

 

Fonts:  If you are using Google Fonts, the @import line must be the first line in your CSS file or the fonts do not work.

 

 

When you see a code block like this with the dots between the tags

<body>
    <div id="content">
        <header>
    . . .
        </footer>
    </div>
</body>

This means that you have to look for the code segment above and below the dots and you insert the new code into your code so that they look the same.

 

 

When the instructions have rules written like this:

.book, .join  

That means you write that as 1 rule like this in your CSS file.

.book, .join {
    color:            #fff;
    background-color: #0443b3;
    padding:          15px 30px;
    margin:           50px 0 0 0;
    border-radius:    5px;
}

 

.rivers-card, .camping-card, .rapids-card   margin: 200px 0;

The rule will look like this

.rivers-card, .camping-card, .rapids-card {
    margin:     200px 0;
    text-align: center;
}

 

Do not add any rules that are not specified in the instructions until after week 7 . Week 7 you will be doing the page layout using display: flex and display: grid. If you have any display: block, display: inline, etc you must remove them as they will mess up the layout in week 7

 

Some of you are putting styling in the HTML file.

in the HTML tag you do something like this.

<img src="images/image.jpg" style="width: 300px;">

That is not allowed in this class and not allowed in almost all web development classes at BYU-Idaho. The school is teaching HTML5 which means that all styling must be separated from the HTML. The HTML file loads the styling from CSS files. There is no exception in this class to this rule. You will not get full points on the assignments when I see that and I do let you remove the and resubmit.

 

You must read all the instructions to make sure you do not miss something . A few of you that I have had zoom calls with have realized how important it is to read all the instructions. They have realized that they did miss a lot of little details that make a big difference in the web page. Common things that students miss in week 4 are missing rules, spelling issues, difference between id/class names in the HTML vs the CSS files. Using the incorrect selector (tag name, class name selector, id name selector).
