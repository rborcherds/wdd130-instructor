Making your site responsive is quite easy. You start it his way

 

Step 1: create a rule that starts like this

@media screen and (max-width: 900px) {


}

 

Step 2: Create the rule names that have display: grid or display: flex in them and set the display to block like this example.

@media screen and (max-width: 900px) {


    header, nav, main, footer {
        display: block;
    } 

}

 

Step 3.  Start the developer tools in the browser and set the page size to less than 900px as in this example

In Google Chrome this will be on the left side of the screen and in FireFox it is on the right side.

Make sure that the screen width is set to a number below what is in the @media line

 

Step 4: Use the developer tools to see what needs to be adjusted and make the adjustments in the CSS file in VSCode. Add the rules and properties to the @media section of the CSS file.

Step 5: Once the site looks good, save and push to GitHub and retest. Make sure to validate the CSS.
