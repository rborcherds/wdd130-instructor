Quite a few student use the Dropdown menu component from W3Schools this week. I personally do not like the way that they have done that code. To me it is more complicated than it needs to be.

 

Here is an example dropdown menu that I created to show you a different way to do them. I never use the way that W3Schools does them in any website I create. Because I am doing a lot of coding in PHP and JavaScript, the list-item setup (ul/li tags) makes it a lot more complicated.

 

//rborcherds.github.io/wdd130-instructor/examples/dropdown-nav.html

I do use the arrow that a lot of you want to use. That icon is technically called a chevron. I got the SVG code from https://heroicons.com/. There are 2 CSS rules that were created to change the color of the lines, set the width and height, change the thickness of the line and to line it up the way I wanted to the text.  The class names are in the SVG code.
