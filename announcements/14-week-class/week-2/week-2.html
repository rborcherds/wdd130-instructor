Everyone will work on the same website the first 10 weeks. It is a fictitious river running company. You will get to do your own site starting in week 5. No exceptions to this.

 

YOU MUST KNOW THIS INFO:

You will be asked for the class section number. To find this click on PEOPLE in the left menu and it has the section number next to all the students names in this class. This number is required for the site plans and the footer of all web pages you create in this class as part of the copyright notice.

 

Now onto this week.

You should start by doing the preparation videos and/or reading assignment and quiz first. Preferably at the start of the week. This will prepare you for the weeks work.

 

You will setup your hosting this week using Git and GitHub. You must follow the instructions exactly for your operating system. If you run into issue you must post the error you are getting into the Discussion Forum so everyone can learn how to fix issues. I recommend that you use your own personal e-mail address when you create your GitHub account. This is something that you will use as a developer your entire career.

 

You must do the practice assignment even though you do not get graded on this. This practice assignment will style the homepage you created in the Setting up your Git and GitHub repository instructions for your GitHub Hosting. You can see an example of this formatting at: //rborcherds.github.io/wdd130-instructor/

Starting this week , all the assignments will build upon each other. All students build the same fictitious company website, Dry Oar. You will need to complete each weeks work to be able to do the next weeks work. This is the first project that lasts till week 9. Every item under each Module has to have a green check mark on the right side to open the next week.

You will continue learning HTML to add to the site and you will start to learn about CSS (Cascading Style Sheets) each week.

You will be introduced too "Site Plans" this week. Site plans are a document that should be created before any web page is created so that you can keep a consistent look and feel to the page. It also describes the type of images, voice, colors, text of the site, etc of the web site.

In the instructions have steps specific for Windows, MAC or Linux machines. If you do not understand what are in the instructions try this page: https://github.com/git-guides/install-git#install-git-on-linux

You will create a homepage that has links to the About Me page, the Dry Oar website, the Dry Oar Site Plan site, and an activity called Positioning with 2 blank links for use in week 7

Once you create the homepage you will publish it to GitHub.

You will then create the site-plan and the CSS file for the site-plan. Make sure you follow the instructions carefully. A lot of students to do not pay attention to the details and miss points because of the minor details. Make sure you modify all the correct CSS and HTML areas.

 

Spiritual Thought/Question for the Week

Take a scripture do want to learn more about.

Go to the Citation Index website that BYU Provo maintains at https://scriptures.byu.edu

This site is split into 3 sections. The left side is all the scriptures showing the King James Version and the Joseph Smith Translation of the Old and New Testament along with the Book of Mormon, Doctrine & Covenants and the Pearl of Great Price. The middle section is where the text of the talks I am asking you to lookup will be displayed. When you first open the website this part of the web page gives you important information. The right side is the citation index where you will lookup the scripture you want to learn more about. There is a Spanish version at https://escrituras.byu.edu.

 

  • Using the navigation on the right site of the web page, find the scripture you want to learn more about and click on it. Some versus will be in multiple sections as part of a set of versus or an individual one.
  • Select a talk that is being displayed and read the section of the talk where that scripture is referenced. You many need to read one or two paragraphs before and/or after to get the full meaning.
  • If you feel so inclined, post what you learned about the scripture you looked up and what you learned.

This is a great activity to do as a Family Home Evening lesson if you have older children or just adults as in a YSA Ward.
