Welcome to Week 5

This week you will read a talk from Elder Uchtdorf.  This is an article to help you think about sharing the gospel online. You will have a few questions to answer

 

You will then be learning about the HTML table element. This table tag is one that used to be used for all web page layouts in the 1994 to 1997 time of the web. It was the only way to get half decent looking page layouts. Back then there were very few graphics being used in the HTML.

 

You will then learn about creating HTML forms. Forms are used anytime you have to get input from the user. Every website has at least 1 form on the site. A simple login page, registration page, sign-up for a newsletter, contact us page are all example of the use of the form tag.

You will then learn about styling the forms. Styling forms is very important. You will learn about the styling for input layouts. In future classes you will also learn to style display forms. The way you style the is different.

 

You will be creating 2 web pages with a form in them.

The first is a quiz form. This is a very basic way that you can create questions and input for a quiz. You will not be processing the quiz in this class as that is for a future class once you have learned JavaScript as well.

The second will be the Contact Us page for the Rafting Site. Make sure to pay close attention to how this is done. You copy part of the HTML from the About Us page and then you edit just 2 items from that and the page layout will have the same header and footer.

You copy from line 1 of the html to the <main> tag and then from the </main> to the <html> tag at the very end. You can copy the entire page and then just delete the content inside the main tag. You then edit the title and meta description tags in the <head< area. Remember that the title is not allowed to be the same on any page of a website and the description must describe what the page is about. Search engines such as Google use that description to display in the results. Google will not show multiple pages with the same title and/or description on the page. That is a big reason why they need to be different.
