Retaking WDD130 at BYU-Idaho Repetindo WDD130 na BYU-Idaho

If you are retaking WDD130, please follow these instructions to set up your computer: Se você está repetindo WDD130, siga estas instruções para configurar seu computador:

  1. Go into GitHub and rename your repository to include "wdd130" and add something descriptive like old or previous such as "wdd130-old". This image shows the entry box and it is found in the repository settings tab / General menu at the top. Acesse o GitHub e renomeie seu repositório para incluir "wdd130" e adicione algo descritivo como "antigo" ou "anterior", como "wdd130-old". Esta imagem mostra a caixa de entrada e ela está na aba de configurações do repositório / menu General no topo.

    Once you have typed the new name, click "Rename" button. Depois de digitar o novo nome, clique no botão "Rename".

  2. Rename the folder on your local computer from wdd130 to match what you renamed the repository to. How you do this will depend on your operating system. Renomeie a pasta no seu computador local de wdd130 para corresponder ao que você renomeou o repositório. Como fazer isso dependerá do seu sistema operacional.

  3. There is one of three things you can do at this point: Há uma de três coisas que você pode fazer neste ponto:

    • Delete the local repository folder and re-clone the renamed repository from GitHub. Excluir a pasta do repositório local e clonar novamente o repositório renomeado do GitHub.
    • Edit the .git/config file in the renamed folder to point to the new repository name. Editar o arquivo .git/config na pasta renomeada para apontar para o novo nome do repositório.
    • Use the git command line to set the new URL for the renamed repository. Usar a linha de comando do git para definir o novo URL do repositório renomeado.

    Here are the instructions for the last two options: Aqui estão as instruções para as duas últimas opções:

    Editing the .git/config file Editando o arquivo .git/config

    Open the .git folder inside the renamed local repository folder. Inside that folder is a file called config. Open that file in a text editor such as VSCode or Notepad. Abra a pasta .git dentro da pasta do repositório local renomeado. Dentro dessa pasta há um arquivo chamado config. Abra esse arquivo em um editor de texto como VSCode ou Notepad.

    Find the line that starts with "url = " and change the URL to match the new repository name on GitHub. Encontre a linha que começa com "url = " e altere o URL para corresponder ao novo nome do repositório no GitHub.

    You can get the new URL by opening the repository on GitHub and copying the URL from the address bar and add .git at the end like the image above. Você pode obter o novo URL abrindo o repositório no GitHub e copiando o URL da barra de endereços e adicionando .git no final, como na imagem acima.

    This process will allow you to keep the old code in sync with GitHub. Este processo permitirá que você mantenha o código antigo sincronizado com o GitHub.

    Using the git command line to set the new URL Usando a linha de comando do git para definir o novo URL

    Open a terminal in VSCode. On Windows you use Ctrl+` and on a MAC you use ⌃` or you can use the menu option VIEW - Terminal. Abra um terminal no VSCode. No Windows, use Ctrl+` e no MAC, use ⌃` ou você pode usar a opção de menu VIEW - Terminal.

    Once the terminal is open, type the following command, replacing the NEW-URL with the new repository URL: Depois que o terminal estiver aberto, digite o seguinte comando, substituindo NEW-URL pelo novo URL do repositório: 

    git remote set-url origin NEW-URL

This link will take you directly to Step 2 Passo 2. Este link levará você diretamente ao Passo 2
