Setup New Computer for coding with GitHub Configurar Novo Computador para programação com GitHub Configurar Nuevo Computador para codificación con GitHub

This page describes how to set up a new computer for a student to keep working with GitHub without having to recreate the code or the repository. Esta página descreve como configurar um novo computador para um estudante continuar trabalhando com o GitHub sem precisar recriar o código ou o repositório. Esta página describe cómo configurar un nuevo computador para que un estudiante continúe trabajando con GitHub sin necesidad de recrear el código o el repositorio.

Setup VSCode Configurar VSCode Configurar VSCode

Install VSCode and the extensions that are require for class. Do not install extra extension as they will not be used and will slow down the startup time. Instale o VSCode e as extensões necessárias para a aula. Não instale extensões extras, pois elas não serão usadas e irão desacelerar o tempo de inicialização. Instale VSCode y las extensiones necesarias para la clase. No instale extensiones adicionales, ya que no se utilizarán y ralentizarán el tiempo de inicio.

Setup Git Configurar Git Configurar Git

Make sure to run the 3 commands for windows or 2 for Mac and Linux Certifique-se de executar os 3 comandos para Windows ou 2 para Mac e Linux Asegúrese de ejecutar los 3 comandos para Windows o 2 para Mac y Linux

  • Type in this configuration command to set the global git username: Digite este comando de configuração para definir o nome de usuário global do git: Escriba este comando de configuración para establecer el nombre de usuario global de git: 

     git config --global user.name "your GitHub username"

    and hit Enter. Nothing will happen if you did it right. If you get an error, you need to fix it. e pressione Enter. Nada acontecerá se você fez certo. Se você obtiver um erro, você precisa corrigi-lo. y presione Enter. Nada ocurrirá si lo hizo correctamente. Si obtiene un error, debe corregirlo.

  • Enter this configuration command to set the email: Digite este comando de configuração para definir o e-mail: Escriba este comando de configuración para establecer el correo electrónico: 

     git config --global user.email "your email@example.com"

    and hit Enter. Nothing will happen if you did it right. If you get an error, you need to fix it. e pressione Enter. Nada acontecerá se você fez certo. Se você obtiver um erro, você precisa corrigi-lo. y presione Enter. Nada ocurrirá si lo hizo correctamente. Si obtiene un error, debe corregirlo.

  • This next command is for Windows only. It does not hurt to also type this command on the MAC and Linux system. Type in this configuration command in order to not ignore case changes in files and folder names: Este próximo comando é apenas para Windows. Não faz mal também digitar este comando no sistema MAC e Linux. Digite este comando de configuração para não ignorar mudanças de maiúsculas/minúsculas em nomes de arquivos e pastas: Este siguiente comando es solo para Windows. No hace daño escribir este comando también en el sistema MAC y Linux. Escriba este comando de configuración para no ignorar los cambios de mayúsculas/minúsculas en los nombres de archivos y carpetas: 

    git config --global core.ignorecase false

    and hit Enter. Nothing will happen if you did it right. If you get an error, you need to fix it. e pressione Enter. Nada acontecerá se você fez certo. Se você obtiver um erro, você precisa corrigi-lo. y presione Enter. Nada ocurrirá si lo hizo correctamente. Si obtiene un error, debe corregirlo.

Finish Setup Finalizar Configuração Finalizar Configuración

Now you need to make any edit and save it. Then go to the Source Control and commit and sync with GutHub. This will force the Git and GitHub authorization logins to take place. Agora você precisa fazer alguma edição e salvá-la. Em seguida, vá para o Source Control e faça commit e sync com o GitHub. Isso forçará os logins de autorização do Git e GitHub a ocorrerem. Ahora necesita hacer alguna edición y guardarla. Luego vaya al Source Control y haga commit y sync con GitHub. Esto obligará a que se realicen los inicios de sesión de autorización de Git y GitHub.
